## Step 1: Installation

```bash
yarn
```

On Ios:

```bash
cd ios && pod install && cd ..
```

## Step 2: Start your Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```

If you can't run by "yarn ios", open XCode and build with it
